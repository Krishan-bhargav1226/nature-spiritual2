## 🚀 DEPLOYMENT (GitHub Pages)

### Step 1 — GitHub पर अपलोड करें
1. GitHub.com पर login करें
2. New Repository बनाएं (जैसे: `nature-spiritual`)
3. सभी files upload करें (folder structure बनाए रखें)

### Step 2 — Pages Enable करें
1. Repository → Settings → Pages
2. Source: **Deploy from a branch**
3. Branch: **main**, Folder: **/ (root)**
4. Save करें

### Step 3 — URL मिलेगा
```
https://yourusername.github.io/nature-spiritual/
```

### Step 4 — Custom Domain (वैकल्पिक)
1. CNAME फ़ाइल बनाएं root में:
   ```
   yourdomain.com
   ```
2. Domain DNS में A records जोड़ें:
   ```
   185.199.108.153
   185.199.109.153
   185.199.110.153
   185.199.111.153
   ```
3. या CNAME record:
   ```
   www → yourusername.github.io
   ```
